PROJECTS

Frontend Development

My Portfolio. The first website I designed, a static website using HTML, CSS, and Bootstrap. It is a simple portfolio page that contains an "About Me", "Projects", and a "Contact Me" section. Overall, the website serves as a good starting point for my portfolio, showcasing my basic skills in web development and my ability to create a simple and functional website. Served as my Casptone 1 Project.

Backend Development

I developed the back-end part of an e-commerce website using fundamental JavaScript concepts to store and manipulate data in variables and arrays. JSON was used to format the data used by applications to store and transport data, while MongoDB, a NoSQL database, was utilized to manage CRUD (Create, Read, Update, Delete) operations. Node.js and Express.js were used to send queries and manipulate data in the database. Through this project, I gained proficiency in the core concepts of JavaScript, data management, and back-end web development. The website was hosted on Render. Served as my Casptone 2 Project.

Full Stack Development

For our final capstone project, we were asked to built a functional e-commerce website using the MERN stack, which includes MongoDB as the database, Express and Node.js on the backend, and React on the frontend. I integrated the backend I developed for my capstone 2 project into this website. This integration allowed me to build a dynamic e-commerce website using React on the front-end that can interact with the database to retrieve and display information. Hosting the application on Vercel allowed for efficient deployment and scaling of the application. Served as my Casptone 3 Project.

This is still a work in progress and it is not yet finished because it is based on my cousin's business, which I plan to offer to her once I am done.
